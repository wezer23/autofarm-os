# 🦞 AutoFarm OS — JARVIS Edition

Система управління фермою **Cherax quadricarinatus** (Австралійський червоноклешневий рак) з JARVIS інтерфейсом у стилі Залізної Людини.

## 🚀 Запуск

```bash
npm install
npm run dev
```

## 🏗️ Deploy на Vercel

1. Завантаж на GitHub
2. Підключи до vercel.com
3. Build: `npm run build`
4. Output: `dist`

## 📱 Функції

- **JARVIS Boot Screen** — анімований запуск
- **Arc Reactor** — живий компонент
- **3 басейни** — Молодь / Вирощування / Товарні
- **Live датчики** — DS18B20, O₂, pH, Аміак
- **Вентиляція** — повзунки приплив/витяжка
- **ШІ протоколи** — 6 нейронних протоколів
- **JARVIS Чат** — розмовний інтерфейс
- **Голосові команди** — Web Speech API
- **Telegram налаштування** — токен + chat ID
- **Калібрування** — offset для кожного датчика
- **GPIO карта** — 8 каналів реле
- **Безпека** — PIN + журнал

## 🌡️ Параметри Cherax quadricarinatus

| Параметр | Оптимум | Попередження | Критично |
|----------|---------|--------------|---------|
| Температура | 27-28.5°C | <24 або >29.5 | <20 або >31 |
| Кисень | >7 mg/L | 5-7 | <5 |
| pH | 7.5-8.5 | 7.0-7.5 | <7.0 або >9.0 |
| Аміак | <0.01 | 0.01-0.05 | >0.05 |
